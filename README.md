# EDT Favorites

Плагин для **1C:Enterprise Development Tools**, упрощающий работу с деревом объектов конфигурации. 
Добавляет две простые кнопки рядом с вкладкой **Навигатор**: **Отображать только измененные объекты** и **Отображать только объекты из избранного**.

## Установка

1. Откройте в EDT **Справка → Установить новое ПО**.
2. Добавьте репозиторий:

   ```text
   https://pavel-aparin.github.io/edt-favorites/
   ```

3. Выберите **EDT Favorites**, завершите установку и перезапустите EDT.

После установки в панели навигатора появятся кнопки фильтрации изменённых и избранных объектов а также кнопка добавления/удаления текущего объекта в избранное.
